# Cyclicpeptide.ai Website

**Domain:** Cyclicpeptide.ai  
**Purpose:** 环肽（Cyclic Peptide）相关的科研/产品/服务网站

## 项目结构

```
cyclicpeptide-website/
├── public/              # 静态资源
│   ├── images/         # 图片
│   ├── docs/           # PDF/文档
│   └── favicon.ico
├── src/
│   ├── components/     # React 组件
│   ├── pages/          # 页面
│   ├── styles/         # CSS/Tailwind
│   └── utils/          # 工具函数
├── package.json
├── next.config.js      # Next.js 配置
└── tailwind.config.js  # Tailwind CSS 配置
```

## 技术栈

- **Framework:** Next.js 14 (React 18)
- **Styling:** Tailwind CSS + shadcn/ui
- **Deployment:** Vercel / Netlify
- **Domain:** Cyclicpeptide.ai
